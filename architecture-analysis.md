# Xuanji 项目架构分析报告

## 1. 项目概览

**Xuanji (璇玑)** 是一个现代化的 AI 编程助手，基于 Claude/OpenAI 等大语言模型，提供 CLI、Electron GUI 和 IM 机器人多种交互方式。项目采用 TypeScript 开发，使用 Ink 框架实现终端 UI，支持 ReAct 推理循环、工具调用、记忆系统、权限控制等核心功能。

### 核心定位
- **开源 AI 编程助手**，类似 Claude Code
- **多适配器架构**，支持 CLI / Electron / IM 机器人
- **安全优先**，双层权限控制（LLM 主动审查 + 硬编码兜底）
- **项目感知**，自动识别项目类型和依赖
- **可扩展**，支持 MCP 协议和 Skill 插件系统

### 技术指标
- **Node.js**: >= 20.0.0
- **TypeScript**: 5.7 (严格模式)
- **代码规模**: 约 50,000+ 行
- **模块数量**: 100+ 核心模块
- **测试框架**: Vitest
- **构建工具**: tsup

## 2. 技术栈

### 运行时与语言
- **Node.js 20+**: 主运行时环境
- **TypeScript 5.7**: 严格类型检查，ESNext 模块系统
- **tsx**: 开发时热加载

### UI 框架
- **Ink 5**: React 18 终端渲染引擎
- **React 18**: 组件化 UI 开发
- **Electron 40**: 桌面应用框架（可选）

### LLM SDK
- **@anthropic-ai/sdk**: Claude 模型支持
- **openai**: OpenAI 模型支持
- **流式响应**: 所有 LLM 调用使用流式 API

### 数据存储
- **better-sqlite3**: 本地 SQLite 数据库（记忆系统）
- **sqlite-vec**: 向量检索扩展
- **JSON5**: 配置文件格式

### 代码分析
- **tree-sitter**: 语法树解析（TypeScript/Python/Java）
- **fast-glob**: 文件搜索
- **ignore**: .gitignore 规则解析

### 工具库
- **diff**: 文件差异计算
- **yaml**: YAML 解析
- **csv-parse**: CSV 数据处理
- **jsdom**: HTML 解析
- **turndown**: HTML 转 Markdown

### 测试与质量
- **Vitest**: 单元测试框架
- **ESLint**: 代码规范检查
- **Prettier**: 代码格式化
