# 南宋三大家诗词对比研究

## 岳飞《满江红》· 李清照《声声慢》· 辛弃疾《定风波》

### 历史坐标
- 靖康之变 (1127年) → 南宋建立
- 岳飞 (1103-1142)：战死/冤杀
- 李清照 (1084-1155)：流亡
- 辛弃疾 (1140-1207)：闲居

### 三种回应
- 岳飞：怒发冲冠 → 向外 → 战
- 李清照：寻寻觅觅 → 向内 → 泣
- 辛弃疾：休恨 → 超越 → 思

### 文学风格
- 岳飞：豪放悲壮
- 李清照：婉约凄切
- 辛弃疾：沉郁顿挫

### 当代启示
- 面对困境的三种姿态：抗争、承受、超越
- 三者互补，共同构成完整的精神图谱
