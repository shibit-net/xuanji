- 智能音箱、汽车、可穿戴设备

- 家用机器人、AR 眼镜

## ⚠️ 诚实面对：核心风险

### 依赖第三方大模型

**风险**：
- 璇玑目前没有自有大模型
- 依赖 Claude/GPT/开源模型
- API 成本、供应商风险

**应对策略**：

1. **短期**：多模型策略 + 本地化
   - 支持 Claude/GPT/Ollama
   - 用户自备 API Key（成本透明）
   - 本地模型完全免费

2. **中期**：深度合作 + 混合架构
   - 批量采购折扣
   - 80% 任务本地处理

3. **长期**：自研管家专用模型
   - Year 5：蒸馏小模型
   - Year 7：自研基座
   - Year 10：多模态管家

**为什么不致命**：
- ✅ 差异化在应用层（记忆、生态），不在模型
- ✅ 多模型比单一模型更韧性
- ✅ 成功先例：Notion、Midjourney 也依赖第三方
