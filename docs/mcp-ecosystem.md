# MCP 生态系统

璇玑深度集成 Model Context Protocol (MCP)，可以轻松连接各种工具和服务。

---

## 什么是 MCP？

**MCP (Model Context Protocol)** 是一个开放协议，用于连接 AI Agent 与外部数据源和工具。

**优势**：
- 标准化接口
- 生态丰富
- 安全可控
- 易于扩展

---

## 内置 MCP 集成

### Playwright 浏览器自动化

**功能**：
- 访问网页
- 点击元素
- 填写表单
- 截图
- 提取内容

**使用场景**：
- 自动访问技术文档网站
- 填写日报/周报
- 网页数据采集
- 自动化测试

**安装方式**：
通过天工坊市场一键安装 Playwright MCP。

---

### 使用 Use Computer

**功能**：
- 模拟鼠标点击和移动
- 模拟键盘输入
- 屏幕截图和 OCR
- 桌面操作

**使用场景**：
- 桌面应用自动化
- 跨平台操作

### 文件系统 MCP

**功能**：
- 读写本地文件

- 文件监听

**已内置**：核心功能自带，无需额外安装。

## Skill 系统

### 什么是 Skill？

Skill 是封装好的高级工作流，比单个 MCP 更强大。

### Skill 与 MCP 的区别

| 维度 | MCP | Skill |
|-----|-----|------|
| 职责 | 单一工具/服务 | 复杂工作流 |
| 粒度 | 小而专 | 大而全 |
| 扩展性 | 协议标准 | 自定义 |
| 复杂度 | 简单 | 可能复杂 |

## 自定义 MCP

### 创建自己的 MCP

如果市场上没有你需要的 MCP，可以自己创建：

1. 遵循 MCP 协议规范
2. 实现工具和资源
3. 测试和调试
4. （可选）分享到市场

### MCP 安装流程

璇玑会自动处理：
- 下载 MCP 包
- 解压和安装依赖
- 生成配置模板
- 注册到系统

## 安全考虑

### MCP 权限

安装新 MCP 前需要确认：
- 需要哪些权限
- 可能的风险
- 是否信任来源

### 权限审计

所有 MCP 的操作都：
- 有日志记录
- 可追溯
- 可撤销
