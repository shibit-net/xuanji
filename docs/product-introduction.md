```bash
# 复制这条命令，粘贴到终端，回车

```

#### Windows 电脑
```powershell
# 复制这条命令，粘贴到 PowerShell，回车
irm https://xuanji.sh/install.ps1 | iex
```

**就这么简单！**

### 2. 配置（填个 API Key）

璇玑需要连接 AI 大脑（Claude 或 ChatGPT）

#### 选择 Claude（推荐）
1. 去 https://console.anthropic.com 注册账号
2. 获取 API Key
3. 在璇玑里输入：
```bash
xuanji config set anthropic.apiKey 你的API密钥
```

#### 选择 ChatGPT
1. 去 https://platform.openai.com 注册
2. 获取 API Key
3. 在璇玑里输入：
```bash
xuanji config set openai.apiKey 你的API密钥
```

#### 完全免费方案（本地运行）
不想花钱买 API？可以用免费的本地 AI：

1. 安装 Ollama（免费软件）
2. 下载免费模型：
```bash
ollama pull llama3
```

3. 使用璇玑：
```bash
xuanji --provider ollama --model llama3
```

**完全免费，完全离线！**

### 3. 开始使用
