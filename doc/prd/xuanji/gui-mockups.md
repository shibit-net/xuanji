## 总结

这些原型图展示了 Xuanji GUI 的核心交互界面，涵盖：

1. ✅ **主界面**：三栏布局（会话列表 + 对话区 + 右侧面板）
2. ✅ **权限确认**：Modal 弹窗，清晰显示风险和参数
3. ✅ **Plan Mode**：可拖拽的任务卡片，分步执行
4. ✅ **保存会话**：自动生成名称、统计信息
5. ✅ **统计面板**：图表可视化 Token 用量和费用趋势
6. ✅ **子代理协作**：多列布局，实时显示各角色进度
7. ✅ **Checkpoint 回滚**：详细影响分析，二次确认
8. ✅ **记忆通知**：轻量级右下角通知
9. ✅ **设置面板**：分组折叠，清晰的表单设计
10. ✅ **工具时间线**：详细的执行历史，支持并行工具分组展示

**设计风格**：
- 深色主题，专业简洁
- 卡片式布局，层次清晰
- 图标 + 文字，易于识别
- 进度条 + 动画，实时反馈
- 键盘快捷键，高效操作
